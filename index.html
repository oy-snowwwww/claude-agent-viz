Guide

Terminal

claude-agents서버 시작 + 브라우저 열기
claude-agents stop서버 종료
claude-agents status현재 상태 확인
claude-agents on세션 시작 시 자동 실행 ON
claude-agents off자동 실행 OFF

Session Tab

클릭 — 해당 세션 필터 (로그, 에이전트)
더블클릭 — 세션 이름 변경
× 버튼 — 종료된 세션 탭 제거
Claude Code 시작/종료 시 탭 자동 생성/제거

Workspace

에이전트 캐릭터가 실시간 상태를 표시
idle = 숨쉬기 애니메이션, working = 바운스 + 발광, done = 초록 글로우
Agent 도구 호출 시 Master ↔ Agent 사이 flyDot 애니메이션

Left Panel

Master 카드 — CLAUDE.md 편집
Agent 카드 — 클릭으로 활성/비활성, 연필 아이콘으로 설정 편집
+ New — 새 에이전트 정의 생성 (~/.claude/agents/)

알림

🔔 버튼 — 브라우저 알림 on/off 토글
탭이 비활성일 때 에이전트 완료, 응답 완료 시 알림
알림 클릭 시 대시보드 탭으로 포커스 이동

Hook

SessionStart → 서버 시작 + 세션 등록
PreToolUse (Agent) → 에이전트 시작 이벤트
PostToolUse (Agent) → 에이전트 완료 이벤트
세션 종료 → 탭 × 버튼 또는 2시간 자동 만료
