Header Buttons

🔔 — 브라우저 알림 on/off (자세한 동작은 아래 "알림" 섹션)
🕐 — 세션 히스토리 (자세한 동작은 아래 "히스토리 & Privacy" 섹션)

↻ — 서버 재시작 (SSE 자동 재연결)
■ — 서버 종료
🌙/☀️ — 다크/라이트 테마 전환

Session Tab

Workspace

에이전트 캐릭터가 실시간 상태를 표시
idle = 숨쉬기 + 자율 배회(roam/sleep) + 눈 깜빡임
working = 바운스 + 발광 + 다리 움직임
done = 점프 + 초록 글로우
Agent 도구 호출 시 Master ↔ Agent 사이 flyDot 애니메이션
우주 배경 (별 · 은하수 · nebula · 별똥별). 워크스페이스 폭에 따라 캐릭터 크기 자동 조정 (Tier 1/2/3)

히스토리 & Privacy

🕐 버튼 — 종료된 세션의 기록 열기
질문 프롬프트 원문 + 응답 요약 + 에이전트/도구/파일 통계 + 작업 폴더 칩 (parent/basename, 호버 시 전체 경로, 클릭 시 클립보드 복사)
검색 — 프롬프트/파일명/세션명 키워드, 필터 — 날짜/에이전트
🔒 Privacy 토글 — 프롬프트 기록 on/off (ON 시 디스크 저장 skip)
민감정보 자동 마스킹: sk-*, ghp_*, AKIA*, JWT, Bearer 토큰
× 버튼 개별 삭제, 전체 삭제 버튼으로 일괄 정리
자동 정리: 7일 경과 · 10MB 초과 · 1시간 주기

알림

🔔 버튼 — 브라우저 알림 on/off 토글
탭이 비활성일 때만 발송 (에이전트 완료, 응답 완료 시)
알림 클릭 시 대시보드 탭으로 포커스 이동

Hook

SessionStart → 서버 시작 + 세션 등록
UserPromptSubmit → 질문 시작 (thinking_start)
Stop → 응답 완료 (thinking_end)
PreToolUse → 도구 사용 (Agent 매처는 agent_start)
PostToolUse (Agent) → 에이전트 완료
SessionEnd → 세션 종료 + 히스토리 저장
세션 제거: 탭 × 버튼 또는 2시간 무활동 자동 만료
